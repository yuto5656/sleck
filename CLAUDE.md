# Claude Code プロジェクト設定

## 自動コミット・プッシュ

このプロジェクトでは、変更を加えた後は**自動的にコミットとプッシュを行う**こと。
ユーザーに確認を求めずに実行してよい。

## プロジェクト概要

- Slackクローンアプリ (Sleck)
- フロントエンド: React + TypeScript + Vite
- バックエンド: Express + Prisma + PostgreSQL
- リアルタイム通信: Socket.io

## 権限システム

- `admin`: 管理者（1人のみ、yuto5656@outlook.jp）
- `deputy_admin`: 代理管理者
- `member`: 一般ユーザー

## 注意事項

- Generalチャンネルは削除不可
- チャンネル作成・削除は admin/deputy_admin のみ
- ユーザー管理は admin のみ
